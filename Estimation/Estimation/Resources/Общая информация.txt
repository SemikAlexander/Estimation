Программа «Обмен валюты» предназначена для ежедневного использования пунктами обмена валют, для улучшения и автоматизации работы. 
В функционал приложения входят:
•  Вывод на рабочий экран программы все выполняемые действия (покупка или продажа валюты)
•  Расчёт «прихода» и «расхода», а также прибыли, которую имеет обменный пункт с учётом различных расходов расходов.
•  Генерация файла формата .docx с детальным отчётом о всех операциях, которые были произведены кассиром, а также расчётом прибыли и расходов.
Программа имеет минималистичный и простой дизайн для удобства работы пользователя с любым уровнем. 
Масштабирование рабочего окна непредусмотренно.
Главное окно состоит из 4-х основных разделов меню, одной таблицы, одной кнопки, 2-х полей ввода и одним выпадающим списком. 

Для заполнения необходимо выбрать валюту из выпадающего списка который подписан "Валюта".
После того как будет выбрана валюта необходимо ввести сумму и курс после чего нажать на кнопку "Добавить".
После этого в таблицу будет добавленна новая строка с входными значениями, которые задал кассир.

Также предусмотрена работа только клавиатурой. После выбора валюты вводится сумма потом можно нажать на клавишу Enter после чего фокус перейдёт на второе текстовое поле.
По заполнению второго текстового поля также нажимем на Enter и фокус переходит на кнопку, после чего можно либо нажать снова Enter или кликнуть правой кнопкой мыши по кнопке и данные будут добавленны в таблицу.

В случае если кассиру нужно удалить какое-то значени из таблицы необходимо дважды нажать правой клавишой мыши на ту ячейку которую пользователь хочет удалить. 
После этого данная ячейка будет удалена.

•  Вкладка "Действие" содержит два пункта "Покупка" и "Продажа". 
В зависимости от типа операции (покупка или продажа валюты) таблица будет заполнена в соответствующие ячейки в таблице.

•  Вкладка "Расходы" представляет собой окно с двумя текстовыми полями и таблицей.
Для добавления расхода необходимо указать сумму, которая была потрачена в текстовом поле под названием "Сумма", а также заполнить второе текстовое поле с описанием на что были потрачены средства, после чего необходимо нажать на кнопку "Добавить".
После этого в таблицу будет добавленна новая строка с входными значениями, которые задал кассир.

Также предусмотрена работа только клавиатурой.

•  Вкладка "Настройка" содержит два пункта "Курсы валют" и "Начало дня". 

	•  Курсы валют - форма, которая позволяет задать текущий курс (покупку и продажу) по каждой валюте которой располагает обменный пункт.

	   Для удобства валюты разбиты на группы.

	   Необходимо в текстовое поле сумму покупки и продажи. 
	   Пустыми текстовые поля быть не должны. В данном случае информация не сохраниться и программа выдаст сообщение об ошибке.
	   После того как каждое текстовое поле будет заполнено необходимо нажать на кнопку "Добавить", после чего данные сохранятся и выведется соответствующее сообщение.

	•  Начало дня - форма, которая позволяет задать по каждой валюте сумму и курс, которым располагает, на начало дня обменный пункт.

	   Для заполнения необходимо выбрать валюту из выпадающего списка который подписан "Валюта".
	   После того как будет выбрана валюта необходимо ввести сумму и курс (данная схема не работает только для рубля, в этом случае необходимо ввести только сумму) и нажать на кнопку "Добавить".
	   После этого в таблицу будет добавленна новая строка с входными значениями, которые задал кассир.

	   Также предусмотрена работа только клавиатурой. После выбора валюты вводится сумма потом можно нажать на клавишу Enter после чего фокус перейдёт на второе текстовое поле.
	   По заполнению второго текстового поля также нажимем на Enter и фокус переходит на кнопку, после чего можно либо нажать снова Enter или кликнуть правой кнопкой мыши по кнопке и данные будут добавленны в таблицу.

	   В случае если кассиру нужно удалить какое-то значени из таблицы необходимо дважды нажать правой клавишой мыши на ту ячейку которую пользователь хочет удалить. 
	   После этого данная ячейка будет удалена.

•  Вкладка "Отчёт" содержит два пункта "Расход/Приход" и "Генерация отчёта". 
	
	•  Расход/Приход - форма, которая позволяет просмотреть по каждой операции (покупка или продажа), а также по валюте все оперцияя а также общую сумму в рублях.

	   Для просмотра информации необходимо выбрать тип оперции из выпадающего списка который подписан "Тип" и валюту из выпадающего списка который подписан "Валюта".
	   После этого в таблицу будет добавленна строки с информацией и внизу формы будет показана общая сумма в рублях.

	•  Генерация отчёта представляет собой функицию генерация документа. 
	   
	   После выбора данного пункта программа произведёт некоторые отчёты, после чего предложит выбрать путь, где сохранить файл с отчётом
